Il contenuto riportato in questo tutorial si riferisce all'interfaccia utente di Qlik Sense Desktop e vi sono alcune differenze rispetto a Qlik Sense. Tuttavia, si dovrebbe essere comunque in grado di completare il tutorial su Qlik Sense. Prima di iniziare il tutorial, è necessario effettuare alcune operazioni preliminari in base all'installazione.  

Con un'installazione di Qlik Sense Desktop verificare che Qlik Sense Desktop sia installato, quindi aprire il PDF e iniziare il tutorial.

Con un'installazione di Qlik Sense chiedere all'amministratore di sistema di: 
1. Fornire accesso all'hub utilizzando QMC.
2. Memorizzare la cartella di origine Tutorial nel computer server.
3. Assicurarsi che nel server sia installato il motore di database Access corretto.
4. Caricare dati.
Sarà quindi possibile aprire il PDF e iniziare il tutorial.
